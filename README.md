# Pomodoro Timer

Simple, lightweight Pomodoro timer to boost focus and manage work/break cycles.

## Features
- Focus and break timers (defaults: 25 / 5 / 15 minutes)
- Start / pause / reset, desktop notifications, sound alerts
- Customizable durations and keyboard shortcuts

## Quick start
1. Clone: `git clone https://github.com/CodeByTinku/Pomodoro-Timer.git`
2. Go to the html file then run the file.

## Usage
- Click **Start** to begin a focus session.
- Short and long breaks follow automatically after cycles.
- Open **Settings** to change durations or enable notifications.

## Configuration
- Defaults: Focus 25 min, Short break 5 min, Long break 15 min.
- Settings persist via the UI; optionally edit `config.json` if the project provides one.
